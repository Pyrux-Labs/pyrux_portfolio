  <action>
Convert each data file from a single-locale export to a locale-keyed export using the pattern from the RESEARCH.md. Import the `Locale` type from `@/i18n/config`.

For each data file, the conversion pattern is:

BEFORE:
```typescript
export const items: ItemType[] = [{ title: "Spanish text", ... }];
```

AFTER:
```typescript
import type { Locale } from "@/i18n/config";

export const items: Record<Locale, ItemType[]> = {
  es: [{ title: "Spanish text", ... }],
  en: [{ title: "English text", ... }],
};
```

Specific file conversions:

   - `getProjectById(id: string, locale: Locale)` -> `items[locale].find(...)`
   - `getFeaturedProjects(locale: Locale)` -> `items[locale].filter(...)`

For ALL helper functions that are exported (getProjectById, getCompanyById, getCreatorById, getFeaturedProjects, getProjectsByTechnology), add a `locale: Locale` parameter and access the correct locale branch: `dataMap[locale]`.
