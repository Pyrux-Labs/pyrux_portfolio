**Step A: Add imports**
```typescript
import { useTranslations } from "next-intl";
```

**Step B: Add hooks at top of component function**
```typescript
const t = useTranslations("ComponentNamespace");  // Namespace matches key in es.json/en.json
const { locale } = useLocale();  // Only if component consumes data files
```

**Step C: Replace hardcoded strings with t() calls**
Replace every hardcoded Spanish string in JSX with the corresponding `t("keyName")` call. The key names must match EXACTLY what was defined in es.json/en.json from Plan 02.
